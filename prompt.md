# 台積電 ILC 化學品槽車轉運站 — PDA 行動管理系統

## 1. 系統架構與導覽功能

### 角色導向登入畫面
App 啟動時，使用者需從 3 個選項中選擇自身角色。選擇角色後，使用者需手動輸入個人工號以決定後續的導覽權限與可用功能。
1. **ILC 值班人員**
2. **ILC 司機**
3. **廠商司機**

### 底部分頁導覽列（依角色切換）
* **ILC 值班人員（值班專用）：** 首頁 / 移轉作業 / 警報通知 / 排班管理 / 設定
* **ILC 司機（司機專用）：** 首頁 / 出工作業 / 進停車場 / 工作紀錄
* **廠商司機（廠商專用）：** 首頁 / 離場作業 / 歷史紀錄

---

## 2. 各畫面功能規格說明

### 畫面 1：登入畫面
* **角色選擇：** 垂直排列的 3 個角色大按鍵，方便手持操作。
* **工號輸入彈出視窗：** 提供文字輸入欄位與虛擬數字鍵盤，供使用者手動輸入個人工號登入。

### 畫面 2：首頁儀表板 (Dashboard)
* **ILC 值班人員視角：**
    * 值班資訊卡：顯示當前值班人員姓名、班別時間與值班狀態。
    * 快速鍵功能區：包含「出工登記」、「入場登記」、「離場登記」與「值班打卡」。
* **ILC 司機視角：** 顯示司機姓名、今日派車（指定任務）、當前任務狀態，以及快速鍵「開始出工」、「回報入場」。
* **廠商司機視角：** 顯示所屬廠商名稱、分配槽體資訊，以及快速鍵「登記離場」。

### 畫面 3：出工開始作業流程（ILC 司機專用）
*情境說明：ILC 司機準備前往台積電廠區（Fab），將灌充完畢的空槽車開回 ILC 轉運站停車場。出發前須使用 PDA 手動輸入相關代碼進行登記。*
* **四步驟循序輸入流程指示器：** `① ILC值班 → ② ILC司機 → ③ 槽體 → ④ 廠區`
    * **第一步：** 手動輸入 ILC 值班人員工號 -> 系統確認姓名與工號（例如：「H5406340」）。
    * **第二步：** 手動輸入 ILC 司機工號 -> 系統確認司機姓名（例如：「林大宏」）。
    * **第三步：** 手動輸入槽體編號 -> 系統確認槽體編號資料（例如：「Tank_B」）。
    * **第四步：** 手動輸入/選擇廠區代碼 -> 系統確認出發前往的廠區（例如：「F18A P1」）。
* **確認送出畫面：** 顯示包含上述所有輸入資料的摘要卡片，並提供「確認送出」與「重新輸入」按鍵。

### 畫面 4：進停車場作業流程（ILC 司機專用）
*情境說明：ILC 司機將槽車從台積電廠區開回轉運站停車場，停妥車輛後，手動輸入資料回報進場並結束該趟任務。*
* **四步驟循序輸入流程指示器：** `① ILC值班 → ② ILC司機 → ③ 槽體 → ④ 停車格`
    * **第一步：** 手動輸入 ILC 值班人員工號 -> 確認值班人員姓名。
    * **第二步：** 手動輸入 ILC 司機工號 -> 確認司機姓名。
    * **第三步：** 手動輸入槽體編號 -> 確認槽體資料。
    * **第四步：** 手動輸入停車格編號 -> 確認停放車位（例如：「P-06」）。
* **確認送出畫面：** 顯示包含進場時間與「任務結束」狀態的摘要。系統會同步於背景發出通知，提示值班人員聯繫對應廠商派司機前來取車。

### 畫面 5：廠商離場作業流程（廠商司機專用）
*情境說明：外包廠商司機抵達轉運站停車場準備接手槽車，開離停車場前須手動輸入資料辦理出場登記。*
* **三步驟循序輸入流程指示器：** `① ILC值班 → ② 廠商司機 → ③ 槽體`
    * **第一步：** 手動輸入 ILC 值班人員工號 -> 確認值班人員姓名。
    * **第二步：** 手動輸入廠商司機工號 -> 確認司機姓名與所屬廠商（例如：「劉家慶 — 長春化工」）。
    * **第三步：** 手動輸入槽體編號 -> 確認槽體資料。
* **確認送出畫面：** 顯示包含累計停放時間、離場時間的摘要卡片，並在送出後將該停車格狀態釋放為「空位」。

### 畫面 6：值班打卡功能（ILC 值班人員專用）
* **上班打卡流程：** 點擊「上班打卡」 -> 手動輸入個人工號 -> 確認打卡時間 -> 將姓名加入「當前在值班人員清單」。
* **下班打卡流程：** 點擊「下班打卡」 -> 手動輸入個人工號 -> 確認打卡時間 -> 將姓名從「當前在值班人員清單」中移除。

## 全局核心邏輯與系統整合

* **資料輸入動態模擬：** 本原型 App 採模擬輸入機制。於各步驟點擊輸入欄位後，系統將提供虛擬鍵盤或常用預設快選選單，確認後即可填入測試資料（Mock Data）並跳轉至下一步驟。
* **標準作業作業流程（SOP）：**
    1.  台積電廠區灌充完畢 -> ILC 司機登記出工（畫面 3）。
    2.  ILC 司機載回轉運站 -> 停妥車位並回報進場（畫面 4）。
    3.  廠商司機到場取車 -> 手動輸入資料辦理離場登記，釋放車位（畫面 5）。
* **雙語介面：** 所有主要文字欄位與標題皆採用繁體中文為主、英文（對應對象與術語）為輔的雙語對照方式呈現。
